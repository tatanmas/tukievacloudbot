Usar cuando necesites crear, actualizar o consultar registros en las bases de Tuki CRM. --- # Tuki CRM Operar las bases de datos de Notion del sistema de gestión de Tuki para leads, tareas y notas. ## Overview Este skill provee scripts para interactuar con 3 bases de datos de Notion: - **Tareas**: `9b6a3ee2-05ab-4e47-bc13-a401de09a900` - Tareas de gestión de Eva - **Notas**: `edfeb377-32fd-4636-bf68-bf6252a04c64` - Notas y documentación - **Leads**: `303ddd5f-c230-8087-9f4b-f6f51c7f5ded` - Captaciones de proveedores ## Scripts ### Crear Tarea ```bash python3 skills/tuki-crm/scripts/notion_task.py task --title "Nombre de tarea" --context "Eva" --status "🍵|✅" ``` Estados disponibles: `🍵` (to-do) o `✅` (done) ### Crear Nota ```bash python3 skills/tuki-crm/scripts/notion_task.py note --title "Título" --content "Contenido" --tipo "Eva" ``` ### Query Leads ```bash python3 skills/tuki-crm/scripts/notion_task.py query_leads ``` ##
